# Amazon Sentiment Analysis

## 📌 Project Overview
This project analyzes Amazon product reviews and classifies them as Positive or Negative using Machine Learning.

## 🚀 Features
- Data preprocessing (cleaning, stopword removal)
- TF-IDF feature extraction
- Linear SVM model
- Accuracy & evaluation metrics
- Confusion matrix visualization
- Custom user input prediction

## 🧠 Technologies Used
- Python
- Scikit-learn
- NLTK
- Pandas, NumPy
- Matplotlib, Seaborn

## 📊 Model Used
Linear Support Vector Machine (SVM)
